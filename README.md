# Install

## [Ubuntu Dotfiles](https://github.com/Ckrenzer/dfm/tree/ubuntu)
```
git clone https://github.com/Ckrenzer/dfm.git && \
cd dfm && \
git checkout ubuntu && \
source transfer.sh
```

## [Windows Dotfiles](https://github.com/Ckrenzer/dfm/tree/windows)
```
git clone https://github.com/Ckrenzer/dfm.git && \
cd dfm && \
git checkout windows
```
